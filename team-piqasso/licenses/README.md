## Licenses Folder

This folder collects the attribution and license notes for code, notebooks, and libraries used by `team-piqasso`.

Files in this folder:

- `ACKNOWLEDGMENTS.md`: credits Alice & Bob and explains what upstream material was adapted.
- `THIRD_PARTY_LICENSES.md`: maps the direct third-party components referenced in this project to their licenses.

Important note:

- This repository snapshot does not contain a top-level upstream `LICENSE` file for the bundled challenge materials under `challenge/`.
- Because of that, this folder preserves attribution and records the licenses we could identify from the code and the libraries directly referenced by `team-piqasso`, while avoiding claims about a missing local upstream license file.
